Getting started
===============

If you want to get started with the ``greenhouse`` web application, please check
out our :ref:`how-to-guides-deploy-docker-compose` or
:ref:`how-to-guides-deploy-manual` guides. Deploying via `Docker Compose`_ is
recommended |---| it is very simple and our configuration is suitable for most
people.
